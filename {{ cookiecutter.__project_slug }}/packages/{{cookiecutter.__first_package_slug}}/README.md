# {{cookiecutter.first_package_name}}

## Develop the package

Run `yarn install` ***in root directory*** once to install dependencies.

- Start TS compiler in watch mode
  ```sh
  yarn dev
  ```

- Lint package
  ```sh
  yarn lint
  # or
  yarn lint:fix
  ```

- Format package
  ```sh
  yarn format
  ```

- Compile package
  ```sh
  yarn compile
  ```
